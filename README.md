# Digimon Story: Time Stranger Evolution Graph

[简体中文](./README_CN.md)

A lightweight, single-file evolution reference for *Digimon Story: Time Stranger*. Search the Field Guide and explore every reachable evolution and de-digivolution path for a selected Digimon.

## Use It

### Open Online

[Launch on GitHub Pages](https://USERNAME.github.io/digimon_evolution_graph/time_stranger.html)

> The GitHub Pages address is a placeholder and will be updated after deployment.

### Use Offline

1. Download [time_stranger.html](./time_stranger.html).
2. Open the downloaded file in a modern browser.

No installation or local server is required. Names and evolution relationships work offline. Portraits are loaded from the internet when available.

## Features

- 475 Digimon, including DLC entries
- Simplified Chinese and English display
- Search by Field Guide number, Chinese name, or English name
- Complete ancestor and descendant evolution graph for the selected Digimon
- Collapsible sidebar and responsive mobile layout
- Dependency-free single HTML file
