# 《数码宝贝物语 时空异客》进化图

[English](./README.md)

一个轻量、单文件的《数码宝贝物语 时空异客》进化查询工具。你可以搜索游戏图鉴，并查看选中数码兽所有可达的进化与退化路线。

## 使用方式

### 在线使用

[通过 GitHub Pages 打开](https://arceus-jia.github.io/digimon_evolution_graph/time_stranger.html?language=zh)

### 离线使用

1. 下载 [time_stranger.html](https://github.com/arceus-jia/digimon_evolution_graph/releases/latest/download/time_stranger.html)。
2. 使用现代浏览器直接打开下载后的文件。

不需要安装依赖，也不需要启动本地服务器。名称和进化关系可以离线查询；联网时会额外加载数码兽头像。

## 功能

- 收录 475 只数码兽，包含 DLC 图鉴
- 支持简体中文和英文显示
- 支持按图鉴编号、中文名或英文名搜索
- 显示选中数码兽完整的上级与下级进化关系
- 显示每只数码兽的中英文属性与类型信息
- 支持通过悬停、键盘聚焦或选中节点查看中英文进化条件
- 侧边栏可收起，并提供清晰的方向按钮和移动端布局
- 不依赖第三方库，应用只有一个 HTML 文件
