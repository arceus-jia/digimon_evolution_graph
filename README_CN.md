# 《数码宝贝物语 时空异客》进化图

[English](./README.md)

一个轻量、单文件的《数码宝贝物语 时空异客》进化查询工具。你可以搜索游戏图鉴，并查看选中数码兽所有可达的进化与退化路线。

## 使用方式

### 在线使用

[通过 GitHub Pages 打开](https://USERNAME.github.io/digimon_evolution_graph/time_stranger.html)

> 当前 GitHub Pages 地址是占位内容，部署后会更新为正式地址。

### 离线使用

1. 下载 [time_stranger.html](./time_stranger.html)。
2. 使用现代浏览器直接打开下载后的文件。

不需要安装依赖，也不需要启动本地服务器。名称和进化关系可以离线查询；联网时会额外加载数码兽头像。

## 功能

- 收录 475 只数码兽，包含 DLC 图鉴
- 支持简体中文和英文显示
- 支持按图鉴编号、中文名或英文名搜索
- 显示选中数码兽完整的上级与下级进化关系
- 侧边栏可收起，支持移动端布局
- 不依赖第三方库，应用只有一个 HTML 文件
